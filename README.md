# Installation
1. Install python3.9
2. Linux:
  1. Move that BugCheck.py to /usr/bin
  2. ```chmod +x Bugcheck.sh ```
  3. ```./ Bugcheck.sh```
3. Windows: Just Double Click on the file
4. Mac OS: NOT TEST.
